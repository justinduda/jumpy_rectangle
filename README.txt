JUMPY RECTANGLE

A JavaScript Canvas browser game. 

Jumpy Rectangle is insprired primarily by the phone app game "Flappy Bird" that recieved a huge wave of popularity about a couple years ago (probably a bit more).

Jumpy Rectange is programmed in JavaScript, using the Canvas ctx library to display graphics. The game is mainly developed in an Object Oriented style; also using a bit of functional programming.

To play, simple open jumpyrectangle.html in a web browser.
